Design Summary

  • View mode: Dual/Single toggle at top of dropdown
  • Dual mode: Two columns for Left/Right input selection
  • Single mode: One list to select the single displayed input
  • Settings button: Opens modal for advanced configuration
  • Enable/Disable: Toggle switch per input - disabled inputs hidden from dropdown
  • Input names: Editable in settings, not in dropdown
  • Default input: Set per-input in settings (shows at startup)
  • Layout sliders: Center gap and side borders in settings
  • No-signal setup: Capture reference images in settings
  • Dropdown is minimal - just select L/R, nothing else
