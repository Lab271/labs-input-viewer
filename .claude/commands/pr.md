Create a pull request for the current branch.

Usage: /pr [title]

This command will:
1. Ensure all changes are committed
2. Push the current branch
3. Create a PR with proper formatting
4. Include a summary of changes

---

I'll help you create a pull request for your current branch.

Let me:
1. Check git status for uncommitted changes
2. Get the branch name and commits
3. Generate a PR title and description
4. Create the PR using GitHub CLI

The PR title will follow conventional commits format based on your branch name.
The description will include:
- Summary of changes (from commit messages)
- Test plan checklist
- Link to any related issues

If you want a specific title, provide it. Otherwise, I'll generate one from the branch name.
